#  Student Activity Tracker

##  Project Overview

The **Student Activity Tracker** is a lightweight front-end web application that helps students track their daily learning activities and monitor progress. Users can mark tasks as completed and view their progress in real time.

---

##  Objective

* Build a structured web application using HTML, CSS, and JavaScript
* Implement interactivity using JavaScript
* Provide real-time progress tracking
* Create a simple and user-friendly interface

---

##  Features

* ✅ Dynamic Activity List
* ✅ Mark Activities as Completed
* ✅ Real-Time Progress Tracking
* ✅ Progress Bar Visualization
* ✅ Data Persistence using LocalStorage
* ✅ Interactive User Interface

---

##  Technologies Used

* HTML5 – Structure
* CSS3 – Styling and Layout
* JavaScript – Logic and Interactivity

##  How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. The application will open in your browser

---

##  How It Works

* Activities are stored in a JavaScript array
* User interactions (checkbox clicks) update activity status
* Progress is calculated dynamically
* Data is saved in LocalStorage to persist after refresh

---

##  Key Functionalities

* Add and display activities
* Toggle activity status (Pending → Completed)
* Automatic progress calculation
* Instant UI updates without page reload
